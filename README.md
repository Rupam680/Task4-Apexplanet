# Task 4: Full Project Implementation

## Objective
The objective of Task 4 is to integrate HTML, CSS, and JavaScript to build a fully functional and interactive web project. This task focuses on implementing real-world features using core front-end technologies.

---

## Description
In this task, a complete web project was developed by combining HTML for structure, CSS for styling and responsiveness, and JavaScript for interactivity. The project demonstrates practical front-end development skills through multiple functional modules.

---

## Features Implemented

### 1. Personal Portfolio Website
- Designed a professional personal portfolio website.
- Includes sections such as About, Projects, and Contact.
- Fully responsive and user-friendly layout.
- Showcases skills and project details.

### 2. To-Do List Application (Using Local Storage)
- Developed a task management feature using JavaScript.
- Allows users to add and manage tasks dynamically.
- Uses browser localStorage to persist data even after page refresh.

### 3. Product Listing Page with Filtering and Sorting
- Created a dynamic product listing interface.
- Displays products using JavaScript DOM manipulation.
- Implemented sorting functionality based on criteria such as price.
- Enhances user interaction and usability.
